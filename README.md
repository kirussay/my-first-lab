# 🚀 Навчальний проєкт: Персональне Портфоліо та React-застосунок

Цей репозиторій є комплексним освітнім проєктом, розробленим у рамках курсу «Основи Frontend-розробки». Він демонструє еволюцію вебсторінки від базової HTML-розмітки до сучасного інтерактивного SPA-застосунку на React.

---

## 📂 Лабораторна робота №1: Структура та Семантика
**Мета:** Створення кістяка сайту за допомогою семантичного HTML5 та налаштування системи Git.

* **Семантика:** Використано теги `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` та `<footer>`.
* **Контент:** Розроблено три сторінки (`index.html`, `about.html`, `contacts.html`).
* **Елементи:** Реалізовано складні таблиці навичок, списки та мультимедіа (`<img>`, `<video>`, `<audio>`).
* **Форми:** Створено форму зворотного зв'язку з валідацією та групуванням полів (`fieldset`).

---

## 🎨 Лабораторна робота №2: Стилізація та Позиціонування
**Мета:** Візуальне оформлення інтерфейсу за допомогою сучасних методів CSS3.

* **Блокова модель:** Налаштовано правильні відступи та `box-sizing`.
* **Позиціонування:** Реалізовано фіксоване меню (`fixed`) та абсолютне позиціонування підписів до фото.
* **Типографіка:** Інтегровано шрифт Google Fonts (Roboto) з використанням відносних одиниць `rem`.
* **Селектори:** Застосовано складні комбіновані селектори та псевдокласи (`:nth-child`, `:hover`, `:focus`).

---

## 📱 Лабораторна робота №3: Адаптивність та Сучасний CSS
**Мета:** Створення "гумового" дизайну за принципом Mobile First.

* **Layout:** Використано **CSS Grid** (`grid-template-areas`) для головної структури та **Flexbox** для дрібних компонентів.
* **Адаптивність:** Налаштовано медіа-запити для планшетів та десктопів.
* **Fluid Typography:** Впроваджено функцію `clamp()` для динамічного масштабування тексту.
* **Змінні:** Вся колірна палітра та відступи винесені в CSS-змінні (`:root`).

---

## 🧠 Лабораторна робота №4: JavaScript Basics & Logic
**Мета:** Опанування базового синтаксису JS, функцій та методів масивів.

* **Логіка:** Реалізовано роботу з функціями (Arrow, Expression), деструктуризацію об'єктів та `optional chaining`.
* **Масиви:** Обробка даних через ланцюжки методів (`filter` -> `map` -> `sort`).
* **Замикання:** Демонстрація роботи лічильників через механізм Closures.

---

## ⚡ Лабораторна робота №5: DOM, Події та API
**Мета:** Створення інтерактивного застосунку (Todo App) та робота з мережею.

* **DOM:** Динамічне створення та видалення елементів списку.
* **API:** Використано **Fetch API** з `async/await` для отримання даних із зовнішнього сервера.
* **Storage:** Реалізовано збереження списку справ та темної теми у `localStorage`.
* **Події:** Оптимізовано обробку дій користувача через **делегування подій**.

---

## ⚛️ Лабораторна робота №6: React Інтерактивний каталог
**Мета:** Перехід до компонентної розробки за допомогою бібліотеки React та інструменту Vite.

* **Архітектура:** Розділення інтерфейсу на функціональні компоненти (`Header`, `MovieCard`, `Search`).
* **Hooks:** Використання `useState` для фільтрації та `useEffect` для імітації завантаження даних.
* **Props:** Організація передачі даних між компонентами.
* **Пошук:** Реалізовано фільтрацію каталогу в реальному часі.

---
