# Лабораторна робота №2: Основи CSS

## Опис проєкту
Це навчальний проєкт — персональне портфоліо, створене за допомогою **HTML5** та **CSS**. Проєкт демонструє навички роботи зі структурою документа, семантичними тегами, таблицями, формами, мультимедіа та сучасною стилізацією з використанням різних селекторів, позиціонування, типографіки та адаптивного дизайну.

## Виконані вимоги:
- [x] Ініціалізація Git-репозиторію та робота з гілками.
- [x] Використання `.gitignore` для чистих комітів.
- [x] Мінімум 10 комітів за стандартом Conventional Commits.
- [x] Семантична розмітка (`<header>`, `<main>`, `<footer>`, `<section>`, `<aside>`).
- [x] Створення таблиць (`<table>`) та складних форм (`<fieldset>`, `<legend>`).
- [x] Робота з мультимедіа (`<img>`, `<video>`, `<audio>`).
- [x] Підключення зовнішніх шрифтів (Google Fonts).
- [x] Використання різних типів селекторів: елементів, класів, ідентифікаторів, комбінованих, атрибутів, псевдокласів та псевдоелементів.
- [x] Позиціонування елементів (`position: fixed`, `relative` + `absolute`).
- [x] Центрування контенту з обмеженням ширини.
- [x] Типографіка з розмірами в `rem`.
- [x] Стилізація форм з ефектами `:focus`.
- [x] Ефект "зебри" для таблиць за допомогою `:nth-child(even)`.
- [x] Горизонтальна навігація без маркерів та підкреслень.

## Структура сайту:
1. **Головна (`index.html`)** — привітання, сектор "hero" та фото автора.
2. **Про мене (`about.html`)** — таблиця навичок з ефектом зебри, відео-презентація та списки.
3. **Контакти (`contacts.html`)** — стилізована форма зворотного зв'язку з валідацією.
4. **Стилі (`styles.css`)** — централізовані стилі для всього сайту.
