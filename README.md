# Лабораторна робота №1: Основи HTML5 та Git

## Опис проєкту
Це навчальний проєкт — персональне портфоліо, створене виключно за допомогою **HTML5** без використання CSS. Проєкт демонструє навички роботи зі структурою документа, семантичними тегами, таблицями, формами та мультимедіа.

## Виконані вимоги:
- [x] Ініціалізація Git-репозиторію та робота з гілками.
- [x] Використання `.gitignore` для чистих комітів.
- [x] Мінімум 10 комітів за стандартом Conventional Commits.
- [x] Семантична розмітка (`<header>`, `<main>`, `<footer>`, `<section>`, `<aside>`).
- [x] Створення таблиць (`<table>`) та складних форм (`<fieldset>`, `<legend>`).
- [x] Робота з мультимедіа (`<img>`, `<video>`, `<audio>`).

## Структура сайту:
1. **Головна (`index.html`)** — привітання та фото автора.
2. **Про мене (`about.html`)** — таблиця навичок, відео-презентація та списки.
3. **Контакти (`contacts.html`)** — форма зворотного зв'язку.
