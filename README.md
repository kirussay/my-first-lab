🚀 Навчальний проєкт: Персональне Портфоліо
Це комплексний проєкт, розроблений у рамках курсу «Основи Frontend-розробки». Проєкт демонструє еволюцію вебсторінки від базової розмітки до сучасного адаптивного дизайну.

📂 Лабораторна робота №1: Структура та Семантика
Мета: Створення кістяка сайту за допомогою семантичного HTML5 та налаштування системи контролю версій Git.

Що було зроблено:
Семантична верстка: Використано теги <header>, <nav>, <main>, <section>, <article>, <aside> та <footer>.

Контент: Створено три сторінки (index.html, about.html, contacts.html).

Списки та таблиці: Реалізовано марковані/нумеровані списки та складну таблицю навичок.

Форми: Створено форму зворотного зв'язку з використанням <fieldset>, <legend> та різних типів input.

Мультимедіа: Інтегровано зображення (<img>), відео (<video>) та аудіо (<audio>) з елементами керування.

Git: Ініціалізовано репозиторій, налаштовано .gitignore та зроблено перші 10 комітів.

🎨 Лабораторна робота №2: Стилізація та Позиціонування
Мета: Перетворення "голого" HTML у візуально привабливий інтерфейс за допомогою CSS3.

Що було зроблено:
Блокова модель: Налаштовано box-sizing: border-box, відступи (margin, padding) та рамки.

Складні селектори: Демонстрація роботи комбінованих, дочірніх, сусідніх та атрибутних селекторів.

Позиціонування: * fixed — для головного меню (хедер завжди зверху).

relative + absolute — для розміщення тексту поверх зображень у <figure>.

Типографіка: Підключено шрифт Roboto через Google Fonts; розміри вказано в одиницях rem.

Візуальні ефекти: Реалізовано ефект "зебри" для таблиць (:nth-child) та стилізовано стани :focus та :hover.

📱 Лабораторна робота №3: Адаптивність та Сучасний CSS
Мета: Створення "гумового" дизайну, який коректно працює на смартфонах, планшетах та ПК.

Що було зроблено:
Mobile First: Весь код спочатку написаний для мобільних пристроїв, а потім адаптований під десктоп.

CSS Grid: * Головна структура сторінки побудована на grid-template-areas.

Автоматична зміна розкладки (з однієї колонки на дві) на великих екранах.

Flexbox: Використано для гнучкого вирівнювання елементів навігації та іконок з використанням gap.

Змінні CSS: Усі кольори та константи винесені в блок :root для легкої підтримки коду.

Fluid Typography: Використано функцію clamp() для динамічного масштабування тексту.

Анімація: Додано @keyframes fadeIn для плавного з'явлення контенту при завантаженні.
